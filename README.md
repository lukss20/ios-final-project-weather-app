Weather Application (iOS)

A full-featured iOS weather application that displays real-time weather data and forecasts using an external API.

Features
- Current weather conditions display  
- 5-day weather forecast  
- Weather icons based on conditions  
- Location-based weather using GPS  
- Tab-based navigation for different views  

Technologies
- Language: Swift  
- Framework: UIKit  
- APIs: Weather API (e.g., OpenWeatherMap)  
- Tools: Xcode  

Key Concepts
- API integration (network requests)  
- JSON parsing and data mapping  
- CoreLocation for GPS access  
- MVC architecture  
- Tab Bar and Navigation Controllers  

Architecture
- ViewControllers manage UI and logic  
- Models represent weather data  
- Network layer handles API requests  

Purpose
The goal of this project was to build a **real-world mobile application**, integrating external APIs, handling asynchronous data, and designing a clean user interface.

Notes
This project serves as a final project demonstrating practical iOS development skills, including networking, UI design, and application architecture.
